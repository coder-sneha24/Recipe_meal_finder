A React-based web application that lets users search for meals based on meal categories (such as Seafood, Dessert, Beef, etc.) using the TheMealDB public API.
The app fetches and displays a list of meals including the meal name, meal ID, and thumbnail image.

This application provides a simple interface where users can:

-Enter a meal category

-Fetch meals using TheMealDB API

View the results in the form of cards displaying:

-Meal name

-Meal ID

-Meal image


 Installation & Setup:-
 1. Clone the repository
    git clone <your-repository-url>

 2. Navigate into the project folder
    cd recipe-search-app

3. Install all dependencies
    npm install

4. Start the development server
    npm start
